作成日: 2026-07-07 ／ 対象: Python 単体テスト 88 件（15 ファイル）＋ C++ gtest 5 件（1 ファイル）
本書は現行テストコードから逆起こししたテスト要件を、サブパッケージ毎に「テスト条件（前提・入力）」と「良品条件（合格判定）」で整理したものである。

1. 概要と実行方法

Python テスト実行python -m pytest（リポジトリルート。pyproject.toml の testpaths が tests/・robot/・simulator/ 配下を収集する）
C++ テスト実行docker コンテナ tomato-harvest-sim-debug 内で colcon build --packages-select franka_ros2_control && colcon test --packages-select franka_ros2_control
テスト環境の前提Python テストは ROS2・Isaac Sim なしで実行可能（rclpy 依存箇所はスタブ/遅延 import で分離）。C++ テストは ROS 2 環境（コンテナは Jazzy）が必要。
全体良品条件Python: 88 passed / 2 skipped（skip は MoveIt2 用 URDF が無い環境のみ）。C++: 100% tests passed。
条件付き skip: 4.5 の test_moveit_service_parameters_include_required_sections と test_moveit_service_writes_ros_params_file は、Isaac Sim 付属の Panda URDF （ISAAC_PANDA_URDF）が存在しない環境では skip される。Isaac Sim 導入済み環境では実行され、pass が良品条件となる。
